r.to.pg

NAME

r.to.pg - exports GRASS raster map to PostgreSQL database into a one column table (raster values only).
(GRASS Raster Program)

SYNOPSIS

r.to.pg
r.to.pg help
r.to.pg rastermap [celltable_name]

DESCRIPTION

The r.to.pg program exports GRASS raster map to PostgreSQL database into a one column table (raster values only). A connection to a PostgreSQL database is needed. The new table will be created in the currently connected database, if the table name does not exist - please check first.

Parameters:

rastermap
Name of an existing raster map to be exported to PostgreSQL.
celltable_name
Name of new table to create in currently connected PostgreSQL database (Default is rastermap name)

EXAMPLE

A PostgreSQl table with one column will be created: 

  column_name   
---------------
   45.2525		
   45.5
   44.124	
   46
with: 
column_name:	Value of rastermap cells

The datatype in the created table is set to float

KNOWN BUGS

The number check of imported values sometimes does not work properly because of some shell problems with 'asterix'(*).
