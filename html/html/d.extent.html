NAME

d.extent - Set window region from currently displayed raster, vector and sites maps with largest map region.
(GRASS Display Program)

SYNOPSIS

d.extent
d.extent help
d.extent [-v]

DESCRIPTION

d.extent can be used to display all objects in currently displayed maps.

OPTIONS

Flag:

-v
Run verbosely.

NOTES

If no raster map is displayed on a monitor, the default region resolution is calculated to fill window by 512 * 512 cells. Otherwise, it will use the minimum resolution of displayed raster maps.

SEE ALSO
