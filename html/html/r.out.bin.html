r.out.bin

NAME

r.out.bin - writes a binary array to standard output
(GRASS Raster Export Program)

SYNOPSIS

r.out.bin [-i] input=name output=name [null=value]

DESCRIPTION

r.out.bin writes a binary array to standard output. Depending on the data type the program will output a integer, float, or double array, unless "-i" is specified, then it will be forced to integer. The region parameters are printed to stderr. This file is compatible with the generic mapping tools (GMT) program xyz2grd if the user wants to convert to netCDF format (see below).

Flag:

-i
Output integer category values, not raster cell values

Parameters:

input
Name of an existing raster map
output
Name of an output binary map (use out=- for stdout)
null
Value to write out for null
Default: 0

AUTHOR

This program is derived from r.out.ascii with a few modifications.
Author: Bob Covill

NOTE

If exported data is moved between platforms byte swapping may be required.

The exported data can be converted to netCDF GMT format with 

xyz2grd .... -ZTLf -F
