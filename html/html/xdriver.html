NAME

XDRIVER - GRASS Monitor - driver to display GRASS maps in X11 environment
(drivers)

DESCRIPTION

The GRASS XDRIVER can be used to display GRASS maps in X11 environment.

Usage in details

  • Set the map size for the XDRIVER:
      export XDRIVER_WIDTH=xxx    (default is 640)
      export XDRIVER_HEIGHT=yyy    (default is 480)
      If "export" command is not accepted by your shell, use "setenv" instead:
      setenv XDRIVER_WIDTH xxx
      setenv XDRIVER_HEIGHT yyy
To control the XDRIVER, use the d.mon module.
