d.area

NAME

d.area - Obtains area/perimeter information on vector polygons
(GRASS Display/Vector Program)

SYNOPSIS

d.area map=name [fillcolor=name] [linecolor=name]

Parameters:

map
Name of existing vector map to be displayed
fillcolor
Color desired for filling polys
Options: red, orange, yellow, green, blue, indigo, white, black, brown, magenta, aqua, gray, grey
Default: white
linecolor
Color desired for drawing map
Default: none
