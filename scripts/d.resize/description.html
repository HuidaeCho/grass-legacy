DESCRIPTION

d.resize sets the current display monitor window size to specified height and width.

EXAMPLE

Resize the display monitor to be the same aspect ratio as a sheet of A4 paper: 
d.mon x0
d.resize w=630 h=891

SEE ALSO

d.info, d.mon, d.monsize
