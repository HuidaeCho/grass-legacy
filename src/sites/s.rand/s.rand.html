NAME

s.rand - Randomly generate a GRASS sites list.
(GRASS Sites Program)

SYNOPSIS

s.rand
s.rand help
s.rand [-r|d] n=value sites=name

DESCRIPTION

s.rand randomly generates sites using the selected random number generator.

OPTIONS

Flags:

-r
Use rand() (the default).
-l
Use drand48().

Parameters:

n=value
Positive integer value indicating the number of sites to be created.
sites=name
Name of a sites file to store random points in.

SEE ALSO

UNIX man pages for rand(3) and drand48(3).
