 This GRASS module calculates the  spectral resonse 
 for raster images at a user specified point (by mouse).

 Requirement:
  gnuplot has to be copied to grass42/bin/g.gnuplot.


 Example: 
    Input: tm1 - tm7 
      ... will display the spectral response in a diagram 
      for each band at the specified location.
