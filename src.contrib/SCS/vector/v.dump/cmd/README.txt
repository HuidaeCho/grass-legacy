Usage:
 v.dump [-falicne] [num=value] map=name[,name,...]

Flags:
  -f   full report [default]
  -a   area report
  -l   line report
  -i   island report
  -c   categories report
  -n   node report
  -e   edit header

Parameters:
  num   Unique number of an area,line,site,category, or node
  map   vector map(s)
