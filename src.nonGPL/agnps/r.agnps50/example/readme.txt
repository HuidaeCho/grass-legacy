Howto install the example?

Go into ypur grass-database: 
 e.g. 
   cd /home/markus/grass_data

Copy the ansi-database (ansi.tgz) into this directory.
Unpack it:
   tar xvfz ansi,tgz

or (if your tar has no z-option)

   gunzip ansi.tgz
   tar xvf ansi.tgz

Now you can start grass.
   location: ansi
   mapset:   PERMANENT


Have fun!
