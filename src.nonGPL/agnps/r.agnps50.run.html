Guidelines for AGNPS-GRASS Interface

r.agnps50.run

AGNPS SCS version 5.00

Supply the following command line arguments:

r.agnps50.run wshed.dat 0 1 1

Where 'wshed.dat' is the name of the data file to be analyzed. and the 0 or 1 determines whether or not AGNPS should produce. a GIS formatted output file. The second 0 or 1 flag is an indicator to turn on the creation of the binary files. The third 0 or 1 flag is to turn on the file creation for the source acct
