***********************************************
Note for SGI IRIX:

You can get the GNU Tools and the egcs compiler
in the IRIX tardist ready-to-install format 
from: 

  http://freeware.sgi.com

***********************************************

The newer GNU c-compiler (gcc 2.8.1 and up) do
no longer support the old 32 bit ABI
(o32 option), so that you must compile in 
the new n32 format. 

The actual distribution (GRASS 5.0beta9) should
compile without problems on IRIX 6.x with
the GNU Compilers (gcc/egcs) and the native
IRIX compiler from SGI.

Compare the file "REQUIREMENTS" for the libraries
that need to be installed in order to link
to.
