The following is a list of environment variables used in GRASS. These
variables were found by searching all *.c *.h files for the words
"setenv" and "getenv" individually. Note that some of them are not yet
found in the source code. These are indicated along with the date they
were last checked (dd/mm/yy).

The second part list GRASS related files.

========================================================================

[v.digit]
  define to click conveniently for two button mouse.
  It's hard to click two buttons simultaneously with two button mouse,
  #ifndef ANOTHER_BUTTON
  #       define MIDDLEB  2
  #       define RIGHTB   3
  #else
  #       define MIDDLEB  3
  #       define RIGHTB   2
  #endif
