8. Feb. 2001

This directory contains the WINDOWS/Cygwin binary code for GRASS 5.0 as of February 2, 2001 in zipped format.

This is the first version with experimental graphical map output.

Installation of binaries:
It is a relative root path so someone can unzip it to their D: drive if necessary. Depending upon where you have your cygwin installed, it could go under that root.

Starting Grass5:
The package includes a StartGrass script in the /usr/local/grass5. This sets the PATH, start the ipc daemon, and spawns an xterm window. From within that window run 'grass5'. You must have WINDOWS/Cygwin from http://sources.redhat.com/cygwin

Get the X11R6.4 binaries from http://sources.redhat.com/cygwin/xfree/

Requirements:
Building IPC for Grass5:
You will need the Inter-process communication (IPC) code: http://www.neuro.gatech.edu/users/cwilson/cygutils/V1.1/cygipc/ These binaries are included in the grass5 zip file; however you will need the .a archive to build on your own.
See See compile_grass_with_wincygnus.txt for more details.

Compiling the GRASS 5 locally - CVS Grass5:
See Grass5 from a version control perspective on http://freegis.org/cgi-bin/viewcvs.cgi/ and http://freegis.org/cgi-bin/viewcvs.cgi/grass/
