int     gvGetErrorCode  (VOID);

This function returns the current error code.  This may be 
called after any gv() function returns an abnormal code to 
determine the current error status.

INPUT
    None
OUTPUT
    None
RETURN
    gvStatus 
		GV_OK
		GV_FAILURE
		GV_MEMORY_ERR
		...
