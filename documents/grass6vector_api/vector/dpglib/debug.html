GRASS 5.0 Vector API: Debug

Debug

Status: Gamma
This module has been designed, reviewed and coded. It should be considered stable.

Debugf is a debuggin function I have used for years. It can be turned on or off simply by setting an environment variable DEBUG. If DEBUG is set to a string, all debuging information will be directed to the file specified by the string. If it is NULL, then debuging information will go to stderr.

Level of debugging information can be controlled by setting the environment varilable DEBUG_LEVEL. DEBUG_LEVEL is a bit mask, where each module can specify what level it will report on. Unsetting DEBUG_LEVEL or setting it to -1 will turn on all levels. Note DEBUG also needs to be set on.

Index of functions

Function Detail

  • void dpgInitDebug (char *file);

    File is the file name of the source file and not a file to open. This function must be called to initialize the debug system.

  • void dpgCloseDebug (VOID);

    This call closes the debug system.

  • int dpgDebugf ARGS ((int level, char *format, ...));

    Use this function to display debugging information. Level is a bit flag which can be controlled by the environment variable DEBUG_LEVEL.
