GRASS 5.0 Vector API: Integer Tree

Integer Tree

Status: Delta
This module has been designed, and coded but is currently under review. It is fairly stable but its interface could change.

I don't know if the design of this module is good or not. But it serves its purpose I guess.

This module is a simpified front end to the tavl library. It hides all the need for callbacks. As such the only data type that it supports is integer keys and a void * pointer to the data.
