GRASS 5.0 Vector API: Linkm Front End

Linkm Front End

Status: Gamma
This module has been designed, reviewed and coded. It should be considered stable.

This library sits on top of the linkm library, and provides extra utility. Here the assumption is 1. that there will be several modules of an application making use of the linkm library and 2. that several of these will possibly be requesting the same size of objects. If you can accept both of those, then this library might be of use to you.

The function dpgLmInit() is simply a plug and play replacement for dpgLinkInit(). And dpgLmRelease() is a similar replacement for dpgLinkCleanup(). It manages a list of sizes of objects and their associated dpgLinkmP tokens. And when a request comes in for a size that it already has, it returns the that token. The effect hopefully is some sharing of these chunks of memory and thus better memory utilization.
