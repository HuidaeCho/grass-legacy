GRASS 5.0 Vector API: Dictionary

Dictionary

Status: Delta
This module has been designed, and coded but is currently under review. It is fairly stable but its interface could change.

The internal code is terribly inefficient and should eventually be rebuilt.

This library implements a simple dictionary. It stores key/value records in a simple colon delimited format. This library is the basis for the Head interface in the Vector API.
