GRASS 5.0 Vector API: Link Memory Library

Link Memory Library

Status: Gamma
This module has been designed, reviewed and coded. It should be considered stable.

This library is useful for fast memory allocation of lost of small objects of the same size. It starts from the assumption that free() is a fairly expensive call, and for things like linked lists and trees, there is considerable overhead involved. This library pre-allocates a chunk of memory and allocates and frees memory from and to that chunk. The chunk grows without calling realloc as needed (but it does not shrink).
