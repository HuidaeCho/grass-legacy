So, what is new in GRASS 4.3? 

The major point is a lot of bugfixing.  For floating point 
support in raster files, please use GRASS 5.0.

In details:

All src.alpha from 4.2.x moved to src  (see below)

New TclTkGRASS in src

Cursor keys working in text screens
