Project ASSIST: GRASS Source Code

This page contains source code produced for use with GRASS. Some of the code is still under development and should be treated as 'alpha' code when compliled.

For information on how to program with C and GRASS, have a look at this tutorial.

All source code (indicated by a '' ) is available in either tarred and compressed, or 'zipped' form. You may have to use the 'Load to Local Disk' option (or equivalent) before selecting.

GRASS Module Description (and details) Tarred and Compressed Zipped
d.param.scale Interactive multi-scale DEM query. d.param.scale.tar.Z dparam.zip
m.in.ntf Ordnance Survey NTF reader. m.in.ntf.tar.Z ntf.zip
r.feat.thin Thins raster surface features to points. r.feat.thin.tar.Z fthin.zip
r.frac.surf Creates a fractal surface using spectral synthesis. r.frac.surf.tar.Z frac.zip
r.gauss.surf Creates a raster of gaussian deviates. r.gauss.surf.tar.Z gauss.zip
r.lags Creates a map of spatial dependency at different lags. r.lags.tar.Z lags.zip
r.moran Creates a map of Moran's I autocorrelation statistics. r.moran.tar.Z moran.zip
r.param.scale Multi-scale DEM analysis. r.param.scale.tar.Z rparam.zip
r.rand.surf Creates a raster of uniform random deviates. r.rand.surf.tar.Z rand.zip
r.statistics Calculates simple univariate statistics from a raster. r.statistics.tar.Z stats.zip
r.xy Creates two rasters of x and y coordinates. r.xy.tar.Z xy.zip
s.delaunay Creates Delaunay triangulation from points. s.delaunay.tar.Z delaunay.zip
v.surf.spline Interpolates a contour map using spline profiles. v.surf.spline.tar.Z spline.zip
